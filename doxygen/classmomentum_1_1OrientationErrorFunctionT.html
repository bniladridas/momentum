Detailed Description

template<typename T>
class momentum::OrientationErrorFunctionT< T >

The OrientationErrorFunction minimizes the F-norm of the element-wise difference of a 3x3 rotation matrix to a target rotation.

Therefore, function f has 9 numbers (FuncDim), and NumVec=3 for the 3 rotation axis.

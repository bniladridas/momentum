Detailed Description

template<typename T>
class momentum::PlaneErrorFunctionT< T >

The PlaneErrorFunction computes the point-on-plane error or point-in-half-plane error.

For point-on-plane error (above = false), it computes the signed distance of a point to a plane using the plane equation. For half-plane error (above = true), the error is zero when the distance is greater than zero (ie. the point being above).

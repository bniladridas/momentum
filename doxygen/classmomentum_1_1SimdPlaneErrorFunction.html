Detailed Description

A highly optimized error function for "point-on-plane" or "point-in-half-plane" errors.

This function is recommended for use only when dealing with a large number of constraints. For a smaller number of constraints, consider using the generic PlaneErrorFunction.

Warning
Due to the multi-threaded evaluation of the error/gradient, the functions are non-deterministic. This might lead to numerical inconsistencies, resulting in slightly different outcomes on multiple calls with the same data.

Detailed Description

template<typename T>
struct momentum::JointStateT< T >

JointState provides all sorts of transformations computed on a joint.

Joint transform is : WorldTransform = ParentWorldTransform * Tz * Ty * Tx * Rpre * R * S, with R depending on rotation order (R = rz * ry * rx). Each joint thus has 7 parameters: 3 translation, 3 rotation, and 1 scale.

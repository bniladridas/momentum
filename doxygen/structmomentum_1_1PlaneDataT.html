Detailed Description

template<typename T>
struct momentum::PlaneDataT< T >

Point-on-plane or Point-in-half-plane constraint data.

The constraint uses the plane equation: x.dot(normal) - d = 0; use -d instead of +d to be backwards compatible with previous implementation.
